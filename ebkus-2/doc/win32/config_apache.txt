Beispiel-Konfiguration des Apache für EBKuS (Windows).
=====================================================

1.

In die Datei httpd.conf des HTTPD-Servers Apache sollten die
untenstehenden Einträge kopiert werden, damit der Apache

a) die User-Authorisierung für EBKuS durchführt.
b) Ein Alias-Eintrag für das EBKuS-Installationsverzeichnis 
   vorhanden ist.
c) Bei den Options ExecCGI eingetragen ist.

Bitte die Reihenfolge der Einträge in der httpd.conf beachten. Siehe
auch die Kommentare und Beispiele in der httpd.conf zu den einzelnen
Anweisungen und das Manual des Apache.

2. 

Mit dem zum Apache gehörenden Programm "htpasswd" die Benutzerkennung
und Passwörter für die Anwender von EBKuS in die bei der Direktive
"AuthUserFile" eingetragene Datei aufnehmen, z.Bsp. efb_users.
hptasswd.exe in einem Dos-Fenster aufrufen, z.Bsp.:

> htpasswd.exe -c efb_users Admin

3. HTTP-Server neu starten!

4. Beispiel zum Kopieren in die httpd.conf des Apache.


################# in die httpd.conf des Apache kopieren ########
# Konfiguration für EBKuS
##############################################################


#
# das Alias-Verzeichnis für EBKuS
#

Alias /efb/ "C:/Programme/ebkus/"  # Alias für das Installationsverzeichnis


#
# Directory Direktiven mit Passwort-Authorisierung für die
# URL http:/localhost/efb/
#

<Directory "C:/Programme/ebkus">        
AllowOverride None
AddHandler cgi-script .py
Options FollowSymLinks ExecCGI Includes  Multiviews # zum Testen noch Indexes
Order Deny,Allow
Deny from all
# Die berechtigte Hostnamen oder IP-Adressen nach from eintragen
Allow from ihr_server.domainname.de	     # ändern!
AuthType Basic
AuthUserFile C:/Programme/Apache Group/Apache/conf/efb_users  # Beispiel.
AuthName "Klientenkartei EFB"
require valid-user
</Directory>


#########Ende der EBKuS Konfiguration#######################

Die Url für die Klientenverwaltung "EBKuS" lautet dann:


http://ihr_server_name/efb/ebcgi.py/menu
